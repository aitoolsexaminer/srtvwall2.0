# srtvwall2.0
Situation Room TV Dashboard
The Situation Room TV Dashboard is a browser-based interface designed for people who want to follow global news in real time from multiple sources at once. Instead of switching between different channels and websites, the dashboard brings everything together into a single screen.

Live TV Monitoring
The dashboard displays a wall of live television channels that can run at the same time. This allows you to monitor multiple broadcasts simultaneously and quickly see how different networks are covering the same story.
Any channel can be selected and moved to the main viewing window with a single click, while the other channels continue playing in the background.
Users have the ability to switch out any default channel by selecting from a diverse set of channels organized by region and topics.

Global News Headlines
Alongside the live broadcasts, the dashboard includes a Global News Monitor that continuously displays the latest headlines from major international news outlets. This provides a quick overview of breaking stories and developing events from around the world.

World Clocks
A built-in world clock panel shows the current time in key cities across different regions. This helps place events in context and makes it easier to follow news as it unfolds in different parts of the world.

Designed for a Big-Screen Experience
The dashboard is designed to work especially well on large displays or multi-monitor setups, creating an experience similar to a newsroom monitoring wall. It offers a clear and organized way to watch multiple broadcasts, track headlines, and stay connected to global events in real time.
